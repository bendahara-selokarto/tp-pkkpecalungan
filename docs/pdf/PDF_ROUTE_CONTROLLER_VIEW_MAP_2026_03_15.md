# PDF Route-Controller-View Map (2026-03-15)

Tujuan: mempermudah audit jalur PDF dengan peta cepat `route -> controller -> view`.

Ringkasan cakupan:

- Total route `GET` di `routes/web.php`: 128
- Total route `report/pdf`: 105
- Total route `*/print`: 3
- Total route `*/print/docx`: 2

Catatan:

- Map ini diambil dari `routes/web.php` + seluruh `*PrintController.php`.
- `DashboardController` punya jalur PDF chart tetapi bukan `*PrintController`.
- `LaporanTahunanPkkPrintController` menghasilkan `docx`, bukan PDF view.
- Dokumen ini dihasilkan oleh `scripts/generate_pdf_route_map.py`.
