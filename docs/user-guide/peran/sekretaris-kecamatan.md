# Panduan Peran: Sekretaris Kecamatan

## Tujuan Halaman Ini

- Anda dapat memonitor kegiatan desa dan menjaga kelengkapan buku administrasi kecamatan.

## Tugas Utama Anda

- Mengelola menu sekretaris kecamatan.
- Melihat progres data pokja dari desa turunan.
- Melihat menu monitoring kegiatan desa.

## Langkah Kerja Harian

1. Buka dashboard untuk melihat ringkasan wilayah kecamatan.
2. Perbarui data pada menu sekretaris yang menjadi tanggung jawab Anda.
3. Gunakan filter bulan pada grafik kegiatan per desa saat butuh fokus periode tertentu.
4. Cek menu pokja sebagai bahan pemantauan.
5. Gunakan menu monitoring desa untuk meninjau detail kegiatan.
6. Cetak laporan bila diperlukan.

## Hasil yang Harus Terlihat

- Ringkasan kegiatan per desa tampil sesuai bulan yang dipilih.
- Buku terisi dan jumlah buku tampil sejajar di dashboard.
- Data monitoring desa bisa dibuka tanpa akses ubah.

## Skenario Sukses

- Anda memilih bulan `Februari`, grafik `Kegiatan per Desa` dan grafik `Jumlah Buku vs Buku Terisi` langsung berubah sesuai bulan aktif.

## Contoh Praktis

- Anda memilih bulan berjalan, mengecek ringkasan per desa, lalu mencetak rekap untuk rapat kecamatan.

## Jika Terkendala

- Dashboard kosong: pastikan desa turunan sudah memiliki data.
- Angka tidak berubah saat ganti bulan: klik ulang filter dan pastikan alamat halaman memuat parameter bulan.
