# Pendahuluan

Panduan ini membantu Anda menyelesaikan tugas **Sekretaris Kecamatan** di aplikasi.

# Navigasi Aplikasi

### Kenali Menu Sekretaris Kecamatan

Menemukan menu kerja utama dan menu pemantauan.

#### Langkah Penggunaan

1. Buka **Dashboard**.
2. Lihat panel kiri **Menu Domain**.
3. Pilih menu di bagian **Sekretaris PKK**.
4. Gunakan menu **Pokja** dan menu **Monitoring Kecamatan** untuk pemantauan.

#### Hasil

Anda mengetahui menu kerja dan menu pemantauan.

# Fitur-Fitur

### Input Data Sekretaris

Menambahkan atau memperbarui data administrasi kecamatan.

#### Langkah Penggunaan

1. Klik salah satu menu sekretaris.
2. Klik **Tambah**.
3. Isi kolom wajib.
4. Klik **Simpan**.

#### Hasil

Data sekretaris tersimpan di tabel.

### Pantau Ringkasan per Desa

Melihat ringkasan kegiatan berdasarkan desa.

#### Langkah Penggunaan

1. Kembali ke **Dashboard**.
2. Pilih **Filter Bulan**.
3. Klik **Terapkan Filter Chart**.
4. Baca grafik **Kegiatan per Desa** dan ringkasan buku.

#### Hasil

Ringkasan per desa tampil sesuai bulan yang dipilih.

### Pantau Data Pokja

Melihat data pokja sebagai bahan evaluasi.

#### Langkah Penggunaan

1. Klik menu **Pokja** yang ingin dipantau.
2. Baca data pada tabel.

#### Hasil

Anda mengetahui kondisi data pokja.

### Gunakan Menu Monitoring Kecamatan

Membaca rekap desa tanpa mengubah data.

#### Langkah Penggunaan

1. Klik **Monitoring Kecamatan**.
2. Pilih rekap yang dibutuhkan.
3. Baca data yang tampil.

#### Hasil

Anda mendapatkan rekap kegiatan desa.

### Cetak Laporan Kecamatan

Mencetak laporan untuk rapat dan arsip kecamatan.

#### Langkah Penggunaan

1. Buka menu yang ingin dicetak.
2. Klik **Cetak**.
3. Simpan file atau lanjutkan cetak.

#### Hasil

Laporan kecamatan tersedia.

# Troubleshooting

### Dashboard Kosong

Pastikan data desa sudah tersedia.

#### Langkah Penggunaan

1. Buka menu sekretaris dan pastikan data tersimpan.
2. Periksa apakah desa turunan sudah ada datanya.
3. Kembali ke **Dashboard** dan klik **Terapkan Filter Chart**.

#### Hasil

**Dashboard** menampilkan ringkasan data.

### Angka Tidak Berubah Saat Ganti Bulan

Pastikan filter sudah diterapkan.

#### Langkah Penggunaan

1. Pilih **Filter Bulan** yang benar.
2. Klik **Terapkan Filter Chart**.
3. Muat ulang **Dashboard** jika diperlukan.

#### Hasil

Ringkasan berubah sesuai bulan yang dipilih.
