# FAQ (Pertanyaan yang Sering Muncul)

## 1. Kenapa menu saya berbeda dengan rekan saya?
Akses menu mengikuti peran, scope, dan wilayah akun Anda. Jika perlu perubahan, minta update ke super admin.

## 2. Kenapa ada label `RO` di beberapa menu?
`RO` artinya menu hanya bisa dibaca. Anda tetap bisa melihat data, tetapi tidak bisa mengubah.

## 3. Kenapa saya tidak bisa menyimpan data?
Kemungkinan:
- akun Anda memang tidak punya izin ubah pada modul itu,
- data wajib belum lengkap,
- format isian belum sesuai.

## 4. Kenapa dashboard saya masih kosong?
Dashboard membaca data dari modul sumber. Jika belum ada input atau filter terlalu sempit, ringkasan bisa kosong.

## 5. Bagaimana cara meminta perubahan akses?
Sampaikan kebutuhan akses ke admin dengan menyebut:
- nama Anda,
- peran yang dibutuhkan,
- wilayah kerja.

## 6. Kenapa grafik saya tidak sama dengan bulan sebelumnya?
Kemungkinan Anda sedang memakai filter bulan yang berbeda. Periksa pilihan bulan dan parameter di URL dashboard.

## 7. Kenapa saya tidak bisa membuka modul di luar pokja saya?
Sistem membatasi akses sesuai tanggung jawab. Ini normal untuk menjaga keamanan data.
