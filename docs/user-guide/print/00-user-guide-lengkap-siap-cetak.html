1. Mulai Cepat

Untuk Siapa

Pengguna baru yang ingin langsung memakai sistem.

Tujuan

Anda bisa login, mengenali menu utama, dan mulai bekerja sesuai peran.

Langkah Singkat

  1. Buka halaman login sistem.
  2. Masukkan email dan kata sandi akun Anda.
  3. Setelah masuk, lihat menu yang muncul di sisi kiri.
  4. Pilih menu sesuai tugas Anda hari ini.

Yang Akan Anda Lihat

  • Dashboard ringkas.
  • Menu domain sesuai peran akun Anda.
  • Tanda RO pada menu yang hanya bisa dibaca.

Jika Terkendala

  • Jika gagal login: pastikan email dan kata sandi benar.
  • Jika menu tidak muncul: hubungi admin untuk cek akses akun.
  • Jika halaman menolak simpan data: kemungkinan modul Anda hanya mode baca.

2. Panduan Peran: Sekretaris Desa

Fokus Peran Anda

  • Mengelola modul sekretaris di level desa.
  • Memantau data Pokja I-IV dalam mode baca.

Yang Bisa Anda Kerjakan

  • Menambah, mengubah, dan mencetak data pada modul sekretaris.
  • Melihat data pada modul pokja untuk kebutuhan pemantauan.

Alur Kerja Harian

  1. Cek dashboard untuk ringkasan kondisi data.
  2. Selesaikan input atau pembaruan data di modul sekretaris.
  3. Tinjau modul pokja untuk memastikan data sudah terisi.
  4. Cetak laporan bila dibutuhkan.

Batasan yang Perlu Diketahui

  • Pada modul pokja, Anda bisa melihat data tetapi tidak bisa mengubah.
  • Jika muncul label RO, artinya menu tersebut hanya untuk baca.

3. Panduan Peran: Sekretaris Kecamatan

Fokus Peran Anda

  • Mengelola modul sekretaris di level kecamatan.
  • Memantau data Pokja I-IV dalam mode baca.
  • Memantau kegiatan desa pada menu monitoring kecamatan.

Yang Bisa Anda Kerjakan

  • Menambah, mengubah, dan mencetak data pada modul sekretaris kecamatan.
  • Melihat data pokja tanpa mengubah data.
  • Melihat detail kegiatan desa untuk monitoring.

Alur Kerja Harian

  1. Buka dashboard untuk melihat ringkasan wilayah kecamatan.
  2. Perbarui data pada modul sekretaris yang menjadi tanggung jawab Anda.
  3. Cek modul pokja sebagai bahan pemantauan.
  4. Gunakan menu monitoring untuk melihat kegiatan desa.
  5. Cetak laporan bila diperlukan.

Batasan yang Perlu Diketahui

  • Modul pokja dan monitoring bersifat baca untuk peran sekretaris.
  • Aksi ubah data ditolak jika menu sedang mode RO.

4. Panduan Peran: Pokja Desa

Fokus Peran Anda

Mengelola data pada bidang pokja sesuai tugas Anda.

Yang Bisa Anda Kerjakan

  • Menambah data baru.
  • Mengubah data yang sudah ada.
  • Mencetak laporan dari modul pokja Anda.

Alur Kerja Harian

  1. Buka menu pokja sesuai bidang Anda.
  2. Isi data baru atau perbarui data yang berubah.
  3. Periksa kembali data sebelum disimpan.
  4. Cetak laporan untuk rekap berkala.

Batasan yang Perlu Diketahui

  • Anda hanya bisa mengakses menu pokja yang sesuai dengan akun Anda.
  • Menu di luar pokja Anda tidak akan muncul.

5. Panduan Peran: Pokja Kecamatan

Fokus Peran Anda

Mengelola data pokja pada level kecamatan sesuai bidang Anda.

Yang Bisa Anda Kerjakan

  • Menambah dan memperbarui data pada modul pokja Anda.
  • Melihat rekap data kecamatan sesuai cakupan bidang.
  • Mencetak laporan modul pokja.

Alur Kerja Harian

  1. Masuk ke modul pokja sesuai bidang akun Anda.
  2. Lakukan input atau pembaruan data.
  3. Cek konsistensi data sebelum final.
  4. Cetak laporan jika diperlukan untuk rapat atau evaluasi.

Batasan yang Perlu Diketahui

  • Akses hanya berlaku untuk bidang pokja yang sesuai dengan role akun Anda.
  • Modul di luar bidang tidak dapat dibuka.

6. Panduan Peran: Super Admin

Fokus Peran Anda

Mengelola akun pengguna dan menjaga keteraturan akses sistem.

Yang Bisa Anda Kerjakan

  • Membuat akun pengguna baru.
  • Memperbarui data akun pengguna.
  • Menghapus akun pengguna yang tidak dipakai.

Alur Kerja Harian

  1. Buka menu Manajemen User.
  2. Periksa data user yang aktif.
  3. Tambah atau ubah akun sesuai kebutuhan organisasi.
  4. Pastikan role, scope, dan wilayah user sudah tepat.

Batasan yang Perlu Diketahui

  • Gunakan perubahan role dengan hati-hati agar akses tidak salah sasaran.
  • Ikuti aturan internal saat mengubah akun dengan hak tinggi.

7. Alur: Kelola Data Harian

Tujuan

Membantu Anda mengelola data rutin dengan alur yang rapi dan konsisten.

Langkah Umum

  1. Pilih menu kerja sesuai peran Anda.
  2. Tambah data baru atau ubah data yang perlu diperbarui.
  3. Periksa kembali isian penting sebelum simpan.
  4. Simpan dan pastikan data muncul di daftar.

Hasil yang Diharapkan

  • Data harian tercatat tepat waktu.
  • Data siap dipakai untuk dashboard dan laporan.

Jika Terkendala

  • Tidak bisa simpan: cek apakah menu Anda mode baca (RO).
  • Data tidak muncul: cek filter halaman dan rentang tanggal.

8. Alur: Filter Dashboard dan Membaca Grafik

Tujuan

Membantu Anda membaca kondisi data dengan cepat melalui dashboard.

Langkah Umum

  1. Buka halaman Dashboard.
  2. Pilih filter yang tersedia sesuai peran Anda.
  3. Baca ringkasan angka terlebih dahulu.
  4. Lanjutkan ke grafik untuk melihat pola data.

Hasil yang Diharapkan

  • Anda memahami kondisi data terbaru tanpa membuka satu per satu modul.
  • Anda bisa menentukan area yang perlu ditindaklanjuti.

Jika Terkendala

  • Grafik kosong: cek apakah data pada modul sumber sudah terisi.
  • Filter tidak muncul: cek hak akses akun Anda.

9. Alur: Cetak dan Ekspor Laporan

Tujuan

Membantu Anda menyiapkan laporan yang siap dibagikan untuk rapat dan evaluasi.

Langkah Umum

  1. Masuk ke modul data yang ingin dilaporkan.
  2. Pastikan data yang akan dicetak sudah benar.
  3. Pilih menu cetak laporan pada modul tersebut.
  4. Simpan atau bagikan hasil laporan sesuai kebutuhan.

Hasil yang Diharapkan

  • Laporan sesuai data terbaru.
  • Format laporan siap pakai untuk kebutuhan administrasi.

Jika Terkendala

  • Laporan tidak muncul: pastikan data dan filter sudah benar.
  • Hasil kurang sesuai: cek kembali data sumber sebelum cetak ulang.

10. FAQ (Pertanyaan yang Sering Muncul)

Kenapa menu saya berbeda dengan rekan saya?

Akses menu mengikuti peran akun Anda. Jika perlu akses tambahan, hubungi admin.

Kenapa ada label RO di beberapa menu?

RO artinya menu hanya bisa dibaca. Anda tetap bisa melihat data, tetapi tidak bisa mengubah.

Kenapa saya tidak bisa menyimpan data?

Kemungkinan:

  • akun Anda memang tidak punya izin ubah pada modul itu,
  • data wajib belum lengkap,
  • format isian belum sesuai.

Kenapa dashboard saya masih kosong?

Dashboard mengambil data dari modul sumber. Jika belum ada input, ringkasan akan terlihat kosong.

Bagaimana cara meminta perubahan akses?

Sampaikan kebutuhan akses ke admin dengan menyebut nama, peran yang dibutuhkan, dan wilayah kerja.
