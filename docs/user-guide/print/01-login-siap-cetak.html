Panduan Masuk ke Sistem

TP PKK Pecalungan | Versi cetak | Pembaruan: 24 Februari 2026

Tujuan

Dokumen ini membantu Anda masuk ke aplikasi dengan langkah singkat dan jelas.

Langkah Masuk

  1. Buka halaman login sistem TP PKK.
  2. Isi email akun Anda.
  3. Isi kata sandi akun Anda.
  4. Klik tombol Log in.

Contoh Tampilan Login

Jika Tidak Bisa Masuk

  • Pastikan email dan kata sandi sudah benar.
  • Periksa tombol Caps Lock pada keyboard.
  • Jika tetap gagal, hubungi admin untuk cek akun Anda.
