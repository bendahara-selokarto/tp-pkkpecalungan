# Alur: Kelola Data Harian

## Tujuan

- Membantu Anda mengelola data rutin dengan alur yang rapi dan konsisten.

## Langkah Praktis

1. Masuk ke menu sesuai tugas hari itu.
2. Klik tambah data atau buka data lama yang ingin diperbarui.
3. Isi kolom wajib terlebih dahulu, lalu lengkapi kolom pendukung.
4. Simpan data.
5. Cek data tersimpan di daftar tabel.

## Hasil yang Diharapkan

- Data harian tercatat tepat waktu.
- Data siap dipakai untuk dashboard dan laporan.

## Contoh Praktis

- Anda membuka menu Buku Agenda, menambah satu catatan hari ini, lalu memastikan data muncul di daftar.

## Jika Terkendala

- Tidak bisa simpan: cek apakah menu Anda mode baca (`RO`).
- Data tidak muncul: cek pencarian/filter tanggal pada tabel.
- Muncul penolakan akses: pastikan Anda membuka menu sesuai peran.
