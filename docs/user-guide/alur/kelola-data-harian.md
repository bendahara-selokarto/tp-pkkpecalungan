# Alur: Kelola Data Harian

## Tujuan
- Membantu Anda mengelola data rutin dengan alur yang rapi dan konsisten.

## Langkah Umum
1. Pilih menu kerja sesuai peran Anda.
2. Tambah data baru atau ubah data yang perlu diperbarui.
3. Periksa kembali isian penting sebelum simpan.
4. Simpan dan pastikan data muncul di daftar.

## Hasil yang Diharapkan
- Data harian tercatat tepat waktu.
- Data siap dipakai untuk dashboard dan laporan.

## Jika Terkendala
- Tidak bisa simpan: cek apakah menu Anda mode baca (`RO`).
- Data tidak muncul: cek filter halaman dan rentang tanggal.
