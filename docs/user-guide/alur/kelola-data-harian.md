# Fitur-Fitur

### Kelola Data Harian

Mencatat dan memperbarui data rutin secara konsisten.

#### Langkah Penggunaan

1. Klik menu kerja sesuai tugas hari itu.
2. Klik **Tambah** atau pilih data yang ingin diubah.
3. Isi kolom wajib.
4. Klik **Simpan**.
5. Pastikan data muncul di tabel.

#### Hasil

Data harian tersimpan dan siap dipakai untuk **Dashboard** serta laporan.

# Troubleshooting

### Tidak Bisa Menyimpan Data

Periksa izin dan kelengkapan isian.

#### Langkah Penggunaan

1. Pastikan menu tidak bertanda **Hanya Baca**.
2. Lengkapi semua kolom wajib.
3. Klik **Simpan** kembali.
4. Jika masih gagal, hubungi **Super Admin**.

#### Hasil

Data tersimpan atau Anda mendapatkan bantuan akses.

### Data Tidak Muncul di Tabel

Periksa filter dan urutan data.

#### Langkah Penggunaan

1. Atur **Pencarian** atau **Filter Tanggal** pada tabel.
2. Muat ulang daftar data.
3. Pastikan data tersimpan pada tanggal yang benar.

#### Hasil

Data terlihat sesuai filter yang digunakan.

### Akses Ditolak

Pastikan Anda membuka menu sesuai peran.

#### Langkah Penggunaan

1. Buka **Profil**.
2. Periksa peran akun.
3. Pindah ke menu yang sesuai dengan peran.
4. Jika perlu akses tambahan, hubungi **Super Admin**.

#### Hasil

Anda menggunakan menu yang sesuai peran.
