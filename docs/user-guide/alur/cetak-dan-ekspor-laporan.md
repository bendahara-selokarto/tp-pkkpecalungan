# Fitur-Fitur

### Cetak dan Ekspor Laporan

Menyiapkan laporan yang siap dibagikan.

#### Langkah Penggunaan

1. Buka menu data yang ingin dilaporkan.
2. Pastikan data sudah benar.
3. Klik **Cetak** atau **Cetak Lampiran** sesuai menu.
4. Tunggu file terbuka.
5. Simpan file atau lanjutkan cetak.

#### Hasil

Laporan tersimpan atau tercetak sesuai data terbaru.

# Troubleshooting

### Laporan Tidak Muncul

Periksa data dan filter sebelum mencetak.

#### Langkah Penggunaan

1. Pastikan data sudah ada di tabel.
2. Periksa **Filter Tanggal** atau **Filter Bulan**.
3. Klik **Cetak** lagi.

#### Hasil

File laporan berhasil dibuat.

### Hasil Tidak Sesuai

Perbarui data sumber lalu cetak ulang.

#### Langkah Penggunaan

1. Kembali ke data sumber.
2. Perbarui data yang salah.
3. Klik **Simpan**.
4. Kembali ke menu laporan dan klik **Cetak**.

#### Hasil

Laporan sesuai data terbaru.

### Akses Ditolak Saat Cetak

Pastikan menu sesuai peran Anda.

#### Langkah Penggunaan

1. Buka **Profil**.
2. Periksa peran akun.
3. Gunakan menu yang sesuai peran.
4. Jika perlu akses tambahan, hubungi **Super Admin**.

#### Hasil

Anda mencetak dari menu yang sesuai peran.
