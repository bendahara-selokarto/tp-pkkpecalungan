# Navigasi Aplikasi

### Melihat Daftar Menu

Mengetahui menu yang bisa Anda akses sesuai peran dan wilayah kerja.

#### Langkah Penggunaan

1. Masuk ke **Dashboard**.
2. Lihat panel kiri **Menu Domain**.
3. Perhatikan tanda **Hanya Baca** jika menu hanya dapat dilihat.
4. Cocokkan menu yang tampil dengan daftar berikut.

Anda mengetahui menu yang tersedia untuk akun Anda.
