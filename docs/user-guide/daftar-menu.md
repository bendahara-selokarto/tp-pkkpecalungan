# Daftar Menu Sistem

Berikut peta menu yang tersedia di sistem. Daftar ini dipakai sebagai acuan saat menyusun panduan per menu.

## Catatan

- Menu yang tampil mengikuti peran, wilayah kerja, dan akses akun Anda.
- Tanda hanya-baca menandakan menu hanya bisa dilihat.
- Beberapa menu hanya muncul untuk peran tertentu (contoh: sekretaris atau pokja tertentu).

## Menu Umum (Desa/Kecamatan)

- Dashboard
- Arsip
- Cetak Lampiran
- Profil

## Menu Super Admin

- Manajemen User
- Management Ijin Akses
- Management Arsip
- Profil
