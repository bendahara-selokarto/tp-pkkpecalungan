# Local Artifacts (Non-Tracked)

Folder ini untuk artefak lokal yang tidak perlu masuk version control.
Contoh: PDF kerja sementara, screenshot ad-hoc, atau bahan audit personal.

Aturan:
- Boleh disimpan untuk kebutuhan kerja lokal.
- Tidak dijadikan acuan canonical lintas tim tanpa sinkronisasi ke dokumen `docs/domain/*`.
- Tidak di-commit ke git.
